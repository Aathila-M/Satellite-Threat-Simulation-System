# 🛰️ Satellite Threat Simulation System

## 📌 Overview
An AI-driven cybersecurity framework for simulating and detecting satellite communication attacks.

## 🚀 Features
- Synthetic attack simulation
- Machine Learning threat detection
- Real-time visualization dashboard
- Risk scoring engine
- Cloud deployment ready

## 🧠 Tech Stack
- Python
- Flask
- Scikit-learn
- Plotly
- MongoDB

## 📊 Attack Types Simulated
- Jamming
- Spoofing
- DoS
- MITM
- Malware Injection

## 📈 Model Performance
Accuracy: 94%+
